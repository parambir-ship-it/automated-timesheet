# automated-timesheet
automated timesheet for use
You can use this URL to access the automated timesheet app: https://parambir-ship-it.github.io/automated-timesheet/
In the "Employees" tab, you need to enter one-time information for each employee, including their salary, biweekly hours, and more. You can then allocate each of the projects they are working on, along with the number of biweekly hours they spend on it, and the dates they are working on that contract.
